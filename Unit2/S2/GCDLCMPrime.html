Find GCD using Euclidean Algorithm

The Euclidean algorithm is an efficient method for computing the greatest common divisor (GCD) of two numbers.

# Python program to find GCD using Euclidean algorithm def gcd_euclidean(a, b): iterations = 0 while b: iterations += 1 a, b = b, a % b return a, iterations num1, num2 = 56, 98 result, iters = gcd_euclidean(num1, num2) print(f"GCD of {num1} and {num2} is {result}") print(f"Iterations required: {iters}")

Find LCM using GCD

This program calculates the Least Common Multiple (LCM) of two numbers using their GCD.

# Python program to find LCM using GCD def gcd(a, b): while b: a, b = b, a % b return a def lcm_using_gcd(a, b): return (a * b) // gcd(a, b) num1, num2 = 12, 18 result = lcm_using_gcd(num1, num2) print(f"LCM of {num1} and {num2} is {result}")

Sieve of Eratosthenes

This algorithm finds all prime numbers up to a given limit efficiently.

# Python program for Sieve of Eratosthenes def sieve_of_eratosthenes(n): is_prime = [True] * (n + 1) is_prime[0] = is_prime[1] = False iterations = 0 for i in range(2, int(n**0.5) + 1): if is_prime[i]: for j in range(i*i, n + 1, i): iterations += 1 is_prime[j] = False primes = [i for i, prime in enumerate(is_prime) if prime] return primes, iterations limit = 1000 primes, iters = sieve_of_eratosthenes(limit) print(f"Prime numbers up to {limit}: {primes}") print(f"Iterations required: {iters}")

Key Points to Remember

  • Divisors of a number are all integers that divide it evenly without a remainder
  • The Euclidean algorithm is efficient for finding GCD with time complexity O(log(min(a,b)))
  • LCM can be calculated using the formula: LCM(a, b) = (a × b) / GCD(a, b)
  • Prime checking can be optimized by checking divisors only up to √n
  • Sieve of Eratosthenes is an efficient algorithm for finding all primes up to a large limit
  • Algorithm efficiency is measured by the number of iterations required

Fast Exponentiation (a^b % m)

Efficiently compute modular exponentials using binary representation

This algorithm efficiently computes a^b mod m by breaking down the exponent into its binary representation. At each step, we square the base and apply the modulus operation. We only multiply when the binary digit is 1.

How Segment Trees Work

  • Segment trees are built by recursively dividing the array into halves until we reach individual elements
  • Each node stores the sum of its segment
  • This allows efficient range sum queries in O(log n) time
  • The tree is built in O(n) time
  • Space complexity is O(4n) for storage

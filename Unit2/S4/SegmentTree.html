Range Query [2,5]

Query Explanation

To calculate the sum of elements from index 2 to 5:

  • Start at the root node [0,5]
  • Check if current segment is within [2,5]
  • Traverse to the right child [3,5] (fully within range)
  • Traverse till the left child [2,2] (fully within range)
  • Combine results from nodes [2,2] and [3,5]
Sum = Node [2,2] + Node [3,5] = 3 + 15 = 18

How Segment Trees Work

A segment tree is a data structure that allows efficient range queries and updates. Each node stores information about a segment of the array. For range sum queries, each node stores the sum of elements in its segment.

The tree is built recursively by dividing the array into halves until we reach single elements.

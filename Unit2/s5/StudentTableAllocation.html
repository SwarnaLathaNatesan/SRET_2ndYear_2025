ASCII Lucky Number Table Allocator

This tool calculates lucky numbers based on the ASCII values of characters in student names, then allocates them to tables based on those numbers.

Why +1 after modulus? The modulus operation (X % N) gives results from 0 to N-1. We add +1 to convert this to a range of 1 to N, which makes more sense for table numbering (there's no "Table 0").

Example: (748 % 15) = 13, then +1 = 14 (Table 14)

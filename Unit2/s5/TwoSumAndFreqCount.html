Two Sum Problem

Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target.

Hash Map (Number → Index)

def two_sum(nums, target):
    # Create a dictionary to store value-index pairs
    num_map = {}
    
    for i, num in enumerate(nums):
        complement = target - num
        
        if complement in num_map:
            return [num_map[complement], i]
        
        num_map[num] = i
    
    return []

Summary: Two Sum Problem

  • Use a hash map (dictionary) to store each number's index
  • For each number, calculate the complement (target - current number)
  • Check if the complement exists in the hash map
  • If found, return the current index and the complement's index
  • Time Complexity: O(n) - We traverse the list once
  • Space Complexity: O(n) - We store n elements in the hash map

Frequency Count Problems

Frequency Count Without Hash Map

Count the frequency of each element in an array without using a hash map.

def frequency_count_without_hash(arr):
    # Sort the array first
    arr.sort()
    
    result = []
    count = 1
    current = arr[0]
    
    for i in range(1, len(arr)):
        if arr[i] == current:
            count += 1
        else:
            result.append((current, count))
            current = arr[i]
            count = 1
            
    result.append((current, count))
    return result

# Example usage
arr = [1, 2, 2, 3, 3, 3, 4, 4, 4, 4]
frequencies = frequency_count_without_hash(arr)
for num, freq in frequencies:
    print(f"Number {num} appears {freq} times")

Frequency Count With Hash Map

Count the frequency of each element in an array using a hash map (more efficient).

def frequency_count_with_hash(arr):
    freq_map = {}
    
    for num in arr:
        if num in freq_map:
            freq_map[num] += 1
        else:
            freq_map[num] = 1
            
    return freq_map

# Example usage
arr = [1, 2, 2, 3, 3, 3, 4, 4, 4, 4]
frequencies = frequency_count_with_hash(arr)
for num, freq in frequencies.items():
    print(f"Number {num} appears {freq} times")

Key Points: Frequency Count

  • Hash map approach is more efficient: O(n) time complexity
  • Without hash map requires sorting first: O(n log n) time complexity
  • Hash maps provide constant time O(1) for insert and lookup operations
  • Always consider trade-offs between time and space complexity

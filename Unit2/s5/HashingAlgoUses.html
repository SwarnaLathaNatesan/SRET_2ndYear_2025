Pattern Hash:

The Rabin-Karp algorithm uses a rolling hash function to efficiently find a pattern in a text. It computes a hash value for the pattern and for each possible substring of the text, comparing them to find matches.

Group Anagrams (Prime Product Hashing)

Anagrams can be grouped by using a hash function that produces the same value for words that are anagrams of each other. Two common approaches:

  • Sorting: Sort the characters of each word and use the sorted string as a key
  • Prime Product: Assign a prime number to each letter and multiply them together to get a unique product for each set of letters

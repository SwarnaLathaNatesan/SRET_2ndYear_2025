Binary Exponentiation

Understanding powers through binary decomposition

Concept Overview

Binary exponentiation is a method for efficiently calculating large powers by breaking down the exponent into its binary representation.

For example, to calculate 2¹⁰⁵, we can break down 105 into its binary components:

105 = 64 + 32 + 8 + 1 = 2⁶ + 2⁵ + 2³ + 2⁰

Therefore, 2¹⁰⁵ = 2⁶⁴ × 2³² × 2⁸ × 2¹

Algorithm Implementation

def binary_exponentiation(base, exponent):
    # Handle negative exponents
    if exponent < 0:
        base = 1 / base
        exponent = -exponent
    
    result = 1
    current_power = base
    
    while exponent > 0:
        # If the current bit is set, multiply result with current_power
        if exponent % 2 == 1:
            result *= current_power
        
        # Square the current power
        current_power *= current_power
        # Move to the next bit
        exponent //= 2
    
    return result

# Calculate 2^105 using binary exponentiation
result = binary_exponentiation(2, 105)
print(result)

Key Points to Remember

Binary exponentiation breaks down the exponent into its binary representation
Only O(log n) multiplications are needed instead of O(n)
Each bit position corresponds to a power of two
We only multiply the powers where the binary digit is 1
This method works for any base, not just 2

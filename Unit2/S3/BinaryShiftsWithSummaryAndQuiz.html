Power of Two: Bit Shifting

Understanding how bit manipulation helps determine if a number is a power of two.

What is a Power of Two?

A power of two is a number of the form 2n where n is an integer. These numbers have a special property in their binary representation.

# Powers of Two:
2^0 = 1 → binary: 0001
2^1 = 2 → binary: 0010
2^2 = 4 → binary: 0100
2^3 = 8 → binary: 1000
2^4 = 16 → binary: 10000

Binary Pattern of Powers of Two

In binary representation, powers of two have exactly one '1' bit followed by zeros:

Decimal Binary Power
1 0001 20
2 0010 21
4 0100 22
8 1000 23
16 10000 24
32 100000 25

Bit Manipulation Approach

We can use bitwise operations to efficiently check if a number is a power of two.

Method 1: Using n & (n-1)

For any number n that is a power of two, n & (n-1) equals 0.

# Python solution using n & (n-1)
def is_power_of_two(n):
    return n > 0 and (n & (n - 1)) == 0

Why This Works

If n is a power of two, it has exactly one '1' bit in its binary representation. Subtracting 1 flips all bits after that '1' bit, including the '1' bit itself. When we AND n with n-1, we get 0.

1

Example: n = 8 (1000 in binary)

n - 1 = 7 (0111 in binary)

n & (n-1) = 1000 & 0111 = 0000 = 0

2

Example: n = 6 (0110 in binary)

n - 1 = 5 (0101 in binary)

n & (n-1) = 0110 & 0101 = 0100 ≠ 0

Method 2: Using Bit Shifting

We can also use bit shifting to check if a number is a power of two by counting the number of '1' bits.

# Python solution using bit shifting
def is_power_of_two(n):
    if n <= 0:
        return False
    count = 0
    while n:
        count += n & 1
        n >>= 1
    return count == 1

Step-by-Step Explanation

1

Powers of two are always positive numbers.

8 > 0 → True

2

Calculate n - 1

Subtract 1 from the number: 8 - 1 = 7

7 in binary: 0111

3

Compute n & (n-1)

Perform bitwise AND operation: 1000 & 0111 = 0000

Result is 0 → Power of two

Key Points to Remember

Binary Pattern

Powers of two have exactly one '1' bit in their binary representation.

n & (n-1) Trick

If n is a power of two, n & (n-1) equals 0.

Positive Numbers Only

Powers of two are always positive (n > 0).

Efficiency

Bit manipulation approach is O(1) time complexity.

Bit Counting

Alternative approach: count the number of '1' bits (should be exactly 1).

Edge Cases

0 is not a power of two, and negative numbers are not powers of two.

    n -= 1
    n |= n >> 1

    n |= n >> 4
    n |= n >> 8

# Problem 3: Count the number of powers of two in a range [a, b]
def count_powers_of_two(a, b):
    count = 0
    n = 1
    while n <= b:
        if n >= a:
            count += 1
        n <<= 1 # Multiply by 2 using left shift
    return count

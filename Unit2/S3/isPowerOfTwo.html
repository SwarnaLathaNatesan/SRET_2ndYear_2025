Power of Two (LeetCode 231)

Check if a number is a power of two using bit manipulation. This relates to exponentials of base 2.

Key Insight

A number n that is a power of two has exactly one bit set to 1 in its binary representation.

  • Examples:
  • 1 = 2⁰ = 0001₂
  • 2 = 2¹ = 0010₂
  • 4 = 2² = 0100₂
  • 8 = 2³ = 1000₂

Bit Manipulation Check

If n > 0 and n & (n - 1) == 0, then n is a power of two.

Explanation: n & (n - 1) clears the rightmost set bit of n. If n has only one set bit, this operation results in zero.

Example

For n = 4:

  • 4 = 0100₂
  • 3 = 0011₂
  • 4 & 3 = 0000₂

Python Code

def isPowerOfTwo(n: int) -> bool:
    return n > 0 and (n & (n - 1)) == 0

This runs in constant time O(1) and is very efficient.

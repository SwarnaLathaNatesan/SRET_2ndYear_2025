Exponentiation: Pow(x, n)

Understanding the efficient calculation of powers

Problem Statement

Implement pow(x, n), which calculates x raised to the power n (i.e., xn).

We need to implement this efficiently using a method called "exponentiation by squaring".

Intuition

The naive approach would multiply x by itself n times, resulting in O(n) time complexity.

Instead, we can use exponentiation by squaring which has O(log n) time complexity.

The key idea is to use the property:

  • xn = (x2)n/2 if n is even
  • xn = x * (x2)(n-1)/2 if n is odd

Solution Code

def myPow(x, n):
    if n == 0:
        return 1
    
    if n < 0:
        x = 1 / x
        n = -n
    
    result = 1
    while n > 0:
        if n % 2 == 1:
            result *= x
        x *= x
        n //= 2
    
    return result

Step-by-Step Explanation

1 Handle the case when n is 0: any number to the power of 0 is 1.
2 If n is negative, convert the problem to calculating (1/x)-n.
3 Initialize result to 1.
4 While n is greater than 0:
5 If n is odd, multiply the result by x.
6 Square x and halve n (using integer division).
7 Return the result.

Key Points to Remember

Time complexity is O(log n) instead of O(n)
Handle negative exponents by using reciprocal
Use the properties of exponents to break down the problem
While loop continues until n becomes 0
When n is odd, we multiply the result by the current base

Understanding loops with fun, interactive examples

For Loop

When you know how many iterations you need

A for loop is used when you know exactly how many times you want to execute a block of code.

üçï Pizza Party Example

You have 6 pizza slices and want to add exactly 3 olive pieces to each slice (using a nested loop).

// For loop structure with nested loop
for (let slice = 0; slice < 6; slice++) {
  console.log("Preparing slice " + slice);
  // Inner loop for olives
  for (let olive = 0; olive < 3; olive++) {
    console.log(" Adding olive " + olive);
    addOlive(slice, olive);
  }
}

While Loop

When you need to loop until a condition changes

A while loop is used when you want to repeat something until a condition is no longer true.

üçø Movie Night Example

You have a bowl of popcorn. You'll keep eating until the bowl is empty.

// While loop structure
let popcornCount = 20; // Starting amount
while (popcornCount > 0) {
  eatPopcorn();
  popcornCount--;
  console.log("Popcorn left: " + popcornCount);
}

Try it: Movie Time

Pieces: 20

For Loop vs. While Loop

Aspect For Loop While Loop
When to use When iterations are known in advance When termination depends on a condition
Initialization Done in the loop structure Done before the loop
Condition checking Before each iteration Before each iteration
Increment/Update Done in the loop structure Done inside the loop body
Risk of infinite loop Lower (if condition is properly set) Higher (if condition never becomes false)

Remember this!

For loops are like counting: "Do this 10 times."

While loops are like waiting: "Keep doing this until something happens."

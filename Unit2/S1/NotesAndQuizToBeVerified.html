Loops: For vs While

Use a for loop when you know how many times to iterate. Use a while loop when you need to loop until a condition is met.

Loop Visualization

For loop (known iterations):

for i in range(5):
    print(i)

While loop (unknown iterations):

while temperature < 100:
    heat()

Functions & Return Values

A function can return only one value, but that value can be a tuple containing multiple values.

Function Return Example

def calculate():
    return 3, True

This returns a tuple: (3, True)

Key Points to Remember

  • Decimal uses digits 0-9, binary uses only 0 and 1
  • ASCII provides numerical representation for characters
  • Use for loops for known iterations, while loops for condition-based looping
  • Functions return one value, which can be a tuple containing multiple values
  • Arrays use contiguous memory and may need reallocation when resizing
  • Stacks follow LIFO (Last-In-First-Out) principle
  • Bitwise operations work directly on bits for efficiency

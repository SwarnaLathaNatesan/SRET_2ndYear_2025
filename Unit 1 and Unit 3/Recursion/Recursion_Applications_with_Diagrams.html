Use case: Fetching employee‚Äìmanager hierarchies or category‚Äìsubcategory trees.

In an Employee table, each row stores id and manager_id. To get the entire reporting chain (all levels of subordinates), databases internally use recursive queries (WITH RECURSIVE in SQL, or recursive stored procedures).

WITH RECURSIVE employee_hierarchy AS (
    SELECT id, name, manager_id
    FROM employees
    WHERE id = 1          -- Start with CEO
    UNION ALL
    SELECT e.id, e.name, e.manager_id
    FROM employees e
    INNER JOIN employee_hierarchy eh
    ON e.manager_id = eh.id
)
SELECT * FROM employee_hierarchy;

    Use case: Listing all files & folders in a directory (like ls -R in Linux or File Explorer search).

    The filesystem is a tree structure. To process all files, the OS recursively enters subdirectories.

    void listFiles(char *dir) {
    for each entry in dir {
        if entry is file:
            print(entry)
        else if entry is directory:
            listFiles(entry);   // recursive call
    }
}

🌳 Difference: Height vs Depth

Depth of a Node: Distance (#edges) from the root to that node.

Height of a Node: Distance (#edges) from that node down to the deepest leaf.

Example:

  • Depth of D = 2 (A→B→D)
  • Height of B = 1 (B→D/E)
  • Height of A (root) = 2 (A→B→D)

🔁 Inorder Traversal

# Core Logic: Traverse left → root → right
# Helps to get sorted order for BST
def inorder(root):
    if root:
        inorder(root.left)
        print(root.val, end=" ")
        inorder(root.right)

🔁 Preorder Traversal

# Core Logic: Visit root → traverse left → traverse right
# Useful for tree copy / expression tree construction
def preorder(root):
    if root:
        print(root.val, end=" ")
        preorder(root.left)
        preorder(root.right)

🔁 Postorder Traversal

# Core Logic: Traverse left → traverse right → visit root
# Useful for deleting/freeing nodes bottom-up
def postorder(root):
    if root:
        postorder(root.left)
        postorder(root.right)
        print(root.val, end=" ")

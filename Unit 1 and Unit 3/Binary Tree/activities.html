Activities — Interactive Exercises
Score: 0/0
Goal: Hands-on micro-activities students can do in 10-15 minutes to internalize concepts.

Activity A — Visual Walkthrough

Give a short array input and ask students to draw caller/stack frames for recursion. Example input: [1,2,3]

Deliverable: Draw the call stack and return values.

Activity B — Role Play

Students pair up: one acts as "root" and others as "nodes"; pass a token to demonstrate preorder/inorder/postorder order.

Activity C — Pseudocode to Code

Provide short pseudocode and ask students to convert to actual function. Encourage meaningful variable names like currentNode, leftSubtree.

Activity D — Time/Space Estimation

Give small function and ask for Big-O estimation. Discuss tradeoffs.

Facilitation Tips

  • Keep groups small (3–4).
  • Use a timer (5–7 minutes per activity).
  • Reward quick sketches with quick feedback and a small token.
